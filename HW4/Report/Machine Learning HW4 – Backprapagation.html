Network initialize:

Explanation:
construct a neural network with specific hidden layer numbers and neuron numbers for each layer. Then store each neuron with dictionary type with some attributes.(weight, output etc.) and return the network.

In the homework, I used the network has 1 input layer with 2 neuron (x, y) and 2-hidden layer with 1 neuron each, and 1 output layer with 2 neuron which means the probability of class 1 and class 2.
The weights and bias are random numbers.

sigmoid and its derivative function:

Explanation:
construct sigmoid and sigmoid derivative function.

forward propagation:

Explanation:
function weight_sum is used to sum up all inputs multiply weights. After summation, return result after passing sigmoid function.

function forward_prop is used to trace every neuron in network and do summation of weights and inputs, then store the output result of each neuron. Last, return the output of the output neuron.

backward propagation:

Explanation:
function cross_entropy_derv is the derivative of cross entropy function, used to compute the error between prediction and real result.

function backward_prop propagate the error (computed by cross entropy) from output layer to input layer. Then, store every neuron’s propagated error.

training

Explanation:
After forward propagation and backward propagation, we know the error of each neuron in the neural network. Then we use these error to update weights and bias, just like doing gradient descend.

I use L2-norm to compute loss.

In the traning process, I found that it’s hard to find a appropriate learning rate for the neural network, that is, the neural network is hard to train with fixed learning rate, sometimes it converge to local minumum and sometimes it converge too slow.

Hence, I used Adamgram to dynamically adjust learning rate. The basic idea is, at first, learning rate will be large, and after some epochs, learning rate gradually become smaller. This method make the network converge successfully, however, sometimes the network still converges to local minumum but the method has improved the result a lot.














































def train(network, dataset, epochs=20):
    time = 0
    e = 1e-9
    for epoch in range(epochs):
        err = 0
        for x, y, label in zip(dataset['x'], dataset['y'], dataset['label']):
#         for x, y, label in dataset:
            outcome = forward_prop(network, [x, y])  
            if label==1:
                labels = [0, 1]
            elif label==0:
                labels = [1, 0]
            
            err += ((labels[0]-outcome[0])**2 + (labels[1]-outcome[1])**2)
            
            backward_prop(network, labels)
            # updates weights
            
            learn = 0.5 
            time+=1
            # learn = 0.010
            for idx in range(len(network)):
                layer = network[idx]
                if idx==0:
                    inputs = [x, y]
                else:
                    inputs = [neuron['out'] for neuron in network[idx-1]]
                    
                for neuron in layer:
                    nums = len(neuron['w'])-1
                    # weights
                    for num in range(len(inputs)):
                        if 'lr' not in neuron:
                            neuron['lr'] = 0.0
                        
                        #Do Adagram    
                        neuron['lr'] = neuron['lr'] + (neuron['err'] * inputs[num])**2
                        
                        neuron['w'][num] += (learn / (e+math.sqrt(neuron['lr'])) * (neuron['err'] * inputs[num]) )
                    # bias
                    neuron['w'][-1] += (learn / (e+math.sqrt(neuron['lr'])) * neuron['err'])
                
        if ((epoch+1) %500) == 0:
            print('>epoch= {0}, learning= {1}, error= {2}'.format(epoch+1, "Adagram", err))
            print_acc(network, dataset)

def print_acc(network, dataset):
    correct = 0
    for x, y, label in zip(dataset['x'], dataset['y'], dataset['label']):
        outcome = forward_prop(network, [x, y])
        if x > y and outcome[0] > outcome[1]:
            correct += 1
        elif x < y and outcome[0] < outcome[1]:
            correct += 1
    print('Accuracy: {0}'.format(correct / 100))
